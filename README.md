# Laboratorio de Métodos Cuantitativos Aplicados a la Gestión

Repositorio académico con notebooks, datasets y material práctico para el estudio y aplicación de métodos cuantitativos en problemas de gestión y toma de decisiones.

---

## Objetivos

- Aplicar herramientas matemáticas y estadísticas a la gestión organizacional
- Analizar y visualizar datos con Python
- Implementar modelos de optimización y programación lineal
- Desarrollar pensamiento analítico para la toma de decisiones

---

## Estructura del repositorio

```
├── notebooks/        # Clases en formato Jupyter Notebook (ordenadas por número)
├── Integradoras/     # Trabajos prácticos integradores
├── DF/               # Datasets utilizados en las clases
└── PPT's/            # Material de apoyo
```

## Requisitos

Para ejecutar los notebooks se necesita Python 3.x con las siguientes librerías:

```bash
pip install jupyter pandas numpy matplotlib scipy sympy
```

También se puede trabajar desde [Google Colab](https://colab.research.google.com/) sin necesidad de instalar nada localmente.

---

## Cómo usar este repositorio

**Opción 1 — Google Colab (recomendado para alumnos)**

1. Abrir el notebook deseado en GitHub
2. Reemplazar `github.com` por `githubtocolab.com` en la URL, o usar el botón "Open in Colab"

**Opción 2 — Local**

```bash
git clone https://github.com/Datso653/Laboratorio-de-metodos-Cuantitativos-Aplicados-a-la-gestion.git
cd Laboratorio-de-metodos-Cuantitativos-Aplicados-a-la-gestion

```
